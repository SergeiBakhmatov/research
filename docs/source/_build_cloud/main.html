async main.Query.resolver_users(self, ids: List[int] | None = None, names: List[str] | None = None) → List[User]

Запрос на получение объектов User.

Arguments:

ids {Optional[List[integer]]} – Айди требуемых пользователей.

names {Optional[List[string]]} – Имена требуемых пользователей.

Returns:

users {List[User]} – Список пользователей, соответствующий аргументам, иначе полный список пользователей.

Пример запроса GraphQL

Заголовок запроса

{“Authorization”: “Bearer TOKEN”}

Где TOKEN - и тд и тп

Запрос с параметрами

query {
  resolverUsers(ids: [12, 13, 14], names: ["Dima", "Volodya"]) {
    #Параметры, которые необходимы в ответе
    id
    name
  }
}

Запрос без параметров

query {
  resolverUsers {
    id
    name
  }
}

Пример ответа

"data": {
    "resolverUsers": [
      {
        "id": 12,
        "name": "Dima"
      },
      {
        "id": 13,
        "name": "Volodya"
      }
    ]
}
